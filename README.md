# 🛒 Ecommerce Website

A simple **Ecommerce web application** that allows users to browse products and simulate an online shopping experience.  
This project was created to practice **web development skills** and build a responsive online store interface.

---

## 📌 Features

- Display products with images, prices, and descriptions
- Add products to shopping cart
- Remove products from cart
- Responsive design for mobile and desktop
- Simple and clean user interface

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactivity and dynamic behavior  
- **Git & GitHub** – Version control and project hosting
